如何解决黑苹果唤醒后 Kernel Panic（AppleHDAHDMI_DPDriver::setPowerState）

2021-01-31 Hackintosh 0 Comments Total: 236(words) Read Time: 1(minutes)

在更新到 Catalina 和 Big Sur之后，很多黑苹果系统睡眠后不能唤醒，而是在唤醒几秒钟后直接panic重启。

日志如下

似乎这个问题只在 Haswell 架构的 CPU 出现。

看到 Tonymacx86也有在讨论这个问题，我采取了一个更简便的方法。如果愿意研究的话可以仔细看看。

解决方法

打开 Opencore Configurator （Clover 类似）

挂载 EFI 分区（下面一栏）


文件 - 打开（选择你的config.plist）

在Device Properties下面PciRoot(0,0x0)/Pci(0x1b,0x0)添加一个键No-hda-gfx,值为00000000（八个零）
可以通过右下角的➕添加

保存文件，重启电脑。试试看休眠是不是不会重启了呢
