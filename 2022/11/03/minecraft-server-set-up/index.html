五分钟搭建 Minecraft 服务器

2022-11-03 Develop java, loafing, website 0 评论 字数统计: 610(字) 阅读时长: 2(分)

教你五分钟白嫖腾讯云开我的世界1.19纯净服务器。

提示：这不是搭建长期的服务器，而是和好友联机而使用的，包含了很多为了节省时间的不规范操作，还请注意

白嫖服务器

打开并登录腾讯云，在试用里选择轻量应用服务器（如图）白嫖链接

系统请选择CentOS

接着按提示完成支付，打开管理面板可以看到新增一台服务器。

等待初始化完成，点击登录打开webshell

安装软件

首先登陆，可以选择免密（非常安全）。

然后，依次输入指令：

启动服务器

首次运行服务器会下载相关软件

这时会提示我们更改EULA

把false改成true，按Ctrl + X退出，再按Y保存。

输入stop关闭
然后我们需要更改服务器配置

online_mode true为正版 false 盗版可以登录
enforce_security_profile 需改成false 否则不可登录
motd 显示在游戏里的服务器简介
其他请按需修改

按Ctrl + X退出，再按Y保存。
此时服务器还不能正常访问，需打开防火墙25565端口。

为了让服务器在我们退出了还可以运行，我们要用screen

跨版本支持

考虑到大部分玩家游戏版本不一致，需安装Viabackward插件。
在服务器命令行输入stop,停掉服务器。
下载最新版本的ViaBackwards和ViavVersion

https://github.com/ViaVersion/ViaVersion/releases/download/4.4.2/ViaVersion-4.4.2.jar
